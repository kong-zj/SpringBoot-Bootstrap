# SpringbootStudy

#### 介绍
大二小学期实践项目（拼团系统）

#### 软件架构
采用Springboot + bootstrap + Mybatis + MySQL 所完成的实践项目


#### 使用说明

1.  下载后更改自己本地的Maven配置
2.  更改配置里MySQL的账号密码
3.  在本地数据库创建对应的表
4.  运行Application后访问localhost:8080即可
